# 流浪狗救助网站项目概述

## 项目名称
给它一个家，让爱终止流浪 - 流浪狗救助网络专题

## 项目目标
通过真实故事引发公众对流浪狗困境的共鸣，科普科学养宠、领养代替购买等正确观念，提供领养、云养等一站式救助渠道，构建可持续的救助社群。

## 页面结构

### 1. 首页 (index.html)
- **导航栏**: 清晰展示各栏目名称
- **Hero区域**: 感人的流浪狗故事轮播
- **救助站点实时地图**: 使用Leaflet地图展示救助站点位置
- **最新动态**: 滚动更新救助进展
- **统计数据**: 救助成果可视化展示
- **快速行动按钮**: 领养、捐赠、云养等入口

### 2. 爱心领养页面 (adoption.html)
- **待领养狗狗展示**: 卡片式布局，包含照片、基本信息、性格描述
- **筛选功能**: 按品种、年龄、性别、地区筛选
- **领养流程指导**: 步骤说明和所需材料
- **成功领养故事**:  Carousel展示
- **在线申请表格**: 简化领养申请流程

### 3. 我们在行动页面 (action.html)
- **救助活动报道**: 图文结合展示最新救助行动
- **政策支持**: 政府相关政策和法规介绍
- **社区行动**: 固定投喂点、救助站建设情况
- **志愿者招募**: 如何参与救助活动
- **合作伙伴**: 合作机构和组织展示

### 4. 弃养图记页面 (abandonment.html)
- **图片故事集**: 展示被弃养狗狗的故事
- **弃养原因分析**: 数据可视化展示
- **责任教育**: 科学养宠知识普及
- **预防措施**: 如何避免弃养行为
- **法律援助**: 相关法律法规介绍

### 5. 论坛问答页面 (forum.html)
- **用户论坛**: 分享救助经验、求助信息
- **问答板块**: 专家在线解答疑问
- **救助指南**: 科学救助流程和养护知识
- **资源分享**: 救助物资、医疗资源信息
- **社区互动**: 点赞、评论、分享功能

## 核心交互功能

### 1. 救助站点地图
- 使用Leaflet地图库实现
- 实时显示救助站点位置
- 点击站点查看详细信息
- 提供导航功能

### 2. 领养申请系统
- 在线填写领养申请表
- 上传相关证明材料
- 实时状态跟踪
- 与救助机构直接沟通

### 3. 论坛互动系统
- 用户注册和登录
- 发布帖子、评论、点赞
- 分类浏览和搜索
- 管理员审核机制

### 4. 数据统计展示
- 救助成果数据可视化
- 使用ECharts.js实现图表
- 实时更新统计数据

## 技术栈
- **前端**: HTML5, CSS3, JavaScript
- **样式框架**: Tailwind CSS
- **动画库**: Anime.js
- **地图**: Leaflet
- **图表**: ECharts.js
- **图片轮播**: Splide.js
- **响应式设计**: 适配各种设备

## 设计原则
- 温暖而专业的视觉风格
- 以大地色调为主，营造亲和力
- 突出真实感人的故事内容
- 简洁直观的用户界面
- 强调互动性和参与感
